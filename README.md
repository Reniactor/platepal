###PlatePal

##Recipe App
This is a web application built using React and Next.js that allows users to search for recipes based on ingredients or meal type, and save their favorite recipes to their account. The app uses the Edamam API to fetch recipe data.

##Features

Search recipes by ingredients or meal type
View detailed recipe instructions and ingredients
Create an account to save favorite recipes
Responsive design for desktop and mobile devices
Accessible to users with disabilities
Technologies Used
React
Next.js
Edamam API
Tailwindcss
Getting Started
To run this app locally, follow these steps:

Clone the repository to your local machine using git clone https://github.com/reniactor/recipe-app.git
Install dependencies using npm install
Create an account on the Edamam Developer portal and get an API key
Add your API key to the .env.local file as NEXT_PUBLIC_EDAMAM_API_KEY=<your-api-key>
Run the app using npm run dev
Navigate to http://localhost:3000 in your browser to view the app
